# adbNative

adbNative is an Android project that extracts and adapts ADB-related components from Shizuku to provide native ADB functionality within a standalone environment.

## Overview

This project focuses specifically on ADB communication logic derived from Shizuku and restructures it for direct integration into adbNative.

## Upstream Project

Portions of this project are derived from:

Shizuku  
https://github.com/RikkaApps/Shizuku

Original project developed by Rikka Apps.

Thank Shizuku :3

## Modifications

The following changes were made compared to upstream:

- Extracted ADB-related components only
- Removed unrelated service layers
- Adjusted build configuration
- Integrated into standalone adbNative structure
